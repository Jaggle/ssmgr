android使用教程

  1. 点击 下载安卓客户端
  2. 最好使用文件管理器打开安装APK,部分机型无法在浏览器中安装
  3. 点击应用右上角+号，扫描帐号右侧的二维码添加帐号信息
  4. 然后点击屏幕右下角的飞机图标连接
  5. 如有问题，看 详细文档

mac使用教程

  1. 点击 下载mac客户端
  2. 安装
  3. 文件管理 -> 应用程序 -> 找到shadowsocks ,右击打开，选确定
  4. 点击右上角状态栏的纸飞机图标 -> 扫描频幕上的二维码
  5. 扫描你账号右侧的二维码连接上网
  6. 点击右上角状态安纸飞机图标 -> 从GFWList更新PAC
  7. 如果无法扫描，请手动录入服务器，端口，密码

windows使用教程

  1. 点击 下载windows客户端
  2. 打开，右击状态栏小飞机图标，选择 使用系统代理、PAC->使用本地PAC, PAC->从GFWList更新本地PAC
  3. 如果客户端有扫描二维码的功能，可以直接扫描你帐号右侧的二维码
  4. 如果无法扫描，请手动录入服务器，端口，密码
  5. 如有问题，看 详细文档

linux使用教程

  1. 点击 下载linux客户端
  2. 下载后 `chmod a+x ss-linux-3.0.0.AppImage` 增加可执行权限，然后双击打开
  3. 在打开的软件中 点击菜单里面的 '连接' -> '添加',这里有很多配置的方式，比如手动、URI、识别二维码，请自己摸索
  4. 然后在系统配置中，打开“网络-网络代理-手动”，在Socks主机一栏写入127.0.0.1,端口写入1080,即可
  5. 如果有问题，请多多百度、谷歌，玩Linux就是需要自己多多折腾，Have Fun!
